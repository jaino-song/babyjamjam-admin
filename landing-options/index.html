랜딩 페이지 컨셉 선택

아래 5가지 옵션을 클릭하여 미리보고 마음에 드는 디자인을 골라주세요.

01

Clean Corporate

브랜드 네이비 컬러를 활용한 깔끔하고 프로페셔널한 기업형 디자인. 기능 카드 그리드와 히어로 섹션.

#기업형 #깔끔함 #네이비
02

Dashboard Preview

CSS 3D 목업 대시보드를 보여주는 구조. 숫자 카운트업과 차트 애니메이션으로 데이터 중심의 인상.

#대시보드 #SaaS #데이터
03

Warm & Caring

산후조리원/케어 서비스에 어울리는 부드러운 색감과 둥근 디자인. 플로팅 카드와 후기 섹션 포함.

#따뜻함 #케어 #부드러움
04

Bento Grid Dark

다크모드 기반의 현대적인 SaaS 느낌. 비대칭 벤토 그리드와 네온 글로우 효과.

#다크모드 #모던 #벤토
05

Scroll Story

스크롤을 따라 "문제 → 해결 → 기능"을 서사적으로 전개. 풀페이지 섹션과 목업 UI.

#스토리텔링 #스크롤 #몰입

원하는 컨셉을 선택 후 알려주시면, Next.js page.tsx로 구현하겠습니다.
